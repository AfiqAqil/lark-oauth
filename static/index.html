🚀 Lark OAuth Integration

This demo shows how to integrate Lark (Feishu) OAuth 2.0 authentication with your application.

✨ What happens when you login:

  • 🔐 Secure OAuth 2.0 authentication with Lark
  • 👤 Retrieve your user profile information
  • 🔄 Automatic token refresh handling
  • 🎯 Redirect to success page with your details

💡 Need help? Check the README.md for setup instructions or visit the API documentation.
